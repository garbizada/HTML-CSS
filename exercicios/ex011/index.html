Reproduzindo áudio

Vamos aprender a reproduzir áudios em HTML5

Mas pode ocorrer de usando essa tag <audio src="" autoplay> não saia áudio no navegador , então podemos usar outra forma como: <audio> e dentro da dessa tag, <source src=""midia/musicaem.mp3,ou midia/musicaem.ogg é tmb midia/musicaem.wav . É pra cada source tem um src e um type diferente, sendo o de mp3(audio/mpeg), o OGG(audio/ogg) é em WAV(audio/wav)

Se mesmo assim não for, infelizemente seu navegador nao suporta áudio clique aqui para baixar o arquivo.
